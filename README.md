# Assem Personal Website

Personal Portfolio Website built using **HTML & CSS** to showcase articles, gallery, features, services, skills,
events, pricing plans, videos, statistics, and contact information in a modern and responsive design.

This project represents a complete front-end website suitable for personal branding,
portfolio presentation, or learning advanced HTML & CSS layouts.

---

## Description

Assem Personal Website is a static portfolio website designed to present personal content such as articles,
images, testimonials, services, team members, skills, events, and pricing plans.
The project focuses on clean UI, organized sections, responsive layout, and practical front-end structure.

---

## Features

* Responsive navigation bar with mega menu
* Landing section with introduction
* Articles section with cards layout
* Image gallery with hover effects
* Features section (Quality, Time, Passion)
* Testimonials section
* Team members with social media links
* Services section with icons
* Skills section with progress bars
* How It Works workflow section
* Events section with countdown layout
* Pricing plans section
* Top videos section
* Statistics section
* Discount request form
* Footer with contact info and gallery

---

## Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Google Fonts (Cairo)
* Normalize.css

## How To Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in any modern browser

No additional setup or dependencies required.

---

## Usage

* Edit content directly from `index.html`
* Replace images inside the `images` folder
* Customize styles from `Assem.css`
* Suitable for learning, practice, or personal portfolio
